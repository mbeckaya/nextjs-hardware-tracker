# nextjs-hardware-tracker

**nextjs-hardware-tracker** is a personal PC hardware management app built with **Next.js**, **Prisma**, **TypeScript** and **Tailwind CSS + DaisyUI**.

It allows users to track their PC components, manage builds, and organize their hardware collection. This project is currently in development.

## Planned Features

- Manage PC hardware components (CPU, GPU, RAM, etc.)
- Create and organize PC builds from components
- Track purchase dates, warranty, and hardware condition
- Optional features for planning upgrades and wishlists
- Basic statistics and recommendations for hardware usage

> Note: This project is a work in progress and the UI/UX is still being developed.
